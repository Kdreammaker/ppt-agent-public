# Assistant Mode Playbook

Assistant Mode is a plan-first review workflow. It creates the same deck plan as Auto Mode, then exposes plan, table of contents, slide content, style, and asset decisions before final build when review is requested.

Public files define checkpoint IDs and summary shape only. Private prompt wording, hidden examples, private ranking, and artifact materialization stay private.

## Required Checkpoints

- intake clarification
- deck plan review
- table of contents review
- slide-by-slide content plan review
- font, palette, layout, and style review
- asset metadata candidate review
- final build continue, revise, or skip decision
- delivery review
