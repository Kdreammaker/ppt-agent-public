HOST AI GUIDE

AI에게 다시 시키기 전에 직접 고칩니다.

텍스트, 위치, 크기, 정렬 같은 작은 수정은 로컬 워크벤치에서 끝내고 host AI는 생성과 큰 방향 판단에 집중하게 합니다.

HTML EDITOR

고정 캔버스가 제품의 중심입니다.

16:9 HTML 슬라이드를 PC 화면 중심에 두고, 슬라이드 레일과 속성 패널은 편집을 돕는 보조 도구로 둡니다.

PDF / PPTX HANDOFF

Export는 honest handoff입니다.

PPT Maker는 현재 편집 상태를 정리해 넘깁니다. 실제 PDF/PPTX 완료 표시는 host AI 결과가 있을 때만 보여줍니다.

PRIVACY / SECURITY

기존 발표자료의 내용이 아니라 스타일만 재사용합니다.

Reference Design Library는 로컬 host AI가 추출한 디자인 레시피만 저장하는 방향입니다. 원본 파일, 슬라이드 본문, 비공개 경로는 제품 서버 저장 대상이 아닙니다.

FAQ

출시 전에도 경계는 명확해야 합니다.

PPT Maker가 AI 작성까지 하나요?

아니요. host AI가 작성과 방향 판단을 맡고 PPT Maker는 편집, 검증, handoff, 계정 경계를 맡습니다.

Export는 바로 파일을 내려주나요?

아니요. 실제 host-AI result reference가 없으면 final 완료처럼 보이지 않습니다.

유료 플랜은 크레딧을 계속 차감하나요?

정상적인 유료 편집 흐름에는 visible per-edit credit을 두지 않습니다. 남용 방지를 위한 내부 fair-use guard는 별도로 둡니다.
